# Employee Management System

## Project Overview
This project demonstrates SQL skills using an Employee Management System database.

## Skills Demonstrated
- Database Design
- SQL DDL (CREATE TABLE)
- SQL DML (INSERT, UPDATE, DELETE)
- JOINs
- GROUP BY
- HAVING
- Aggregate Functions
- Window Functions
- Subqueries

## Tables
- Departments
- Employees
- Salaries

## Sample Reports
- Employees by Department
- Highest Paid Employees
- Department Salary Summary
- Employees Joined in the Last Year

## Technologies
- SQL
- Relational Database Concepts
